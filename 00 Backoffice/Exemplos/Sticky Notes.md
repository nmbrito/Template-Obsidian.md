```markdown
> [!STICKY|
> s-30, s-35, s-40, s-45, s-50, s-55, s60    -> Sizes
> yellow, green, blue, purple, aqua, orange  -> Colors
> aside                                      -> Aside text
> left, right, center                        -> Alignment
> title                                      -> Bold title
> ]
> The default sticky 
> (takes default color from Style Settings).
> Size 30
```
